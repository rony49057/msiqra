Put your images in this folder.

Recommended file names for current setup:
- logo.png
- hero.jpg
- about.jpg
- project-road.jpg
- project-building.jpg
- project-school.jpg
- gallery-01.jpg
- gallery-02.jpg
- gallery-03.jpg
- gallery-04.jpg
- gallery-05.jpg
- gallery-06.jpg

IMPORTANT:
This website uses plain HTML/CSS/JS.
Because of browser limitations, the gallery cannot automatically read all files from the folder.
So after adding image files, open script.js and update the image names inside:
- projectData
- galleryData

You can add as many gallery images as you want by copying one gallery object and changing:
- title
- caption
- image path
